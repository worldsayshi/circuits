
## Getting started

Checkout:
```
git clone git@github.com:worldsayshi/WebCola.git
git clone git@github.com:worldsayshi/circuits.git
```
Install:
```
(cd webcola && npm i)
(cd ciscuits && npm i) 
```
Get webcola fix:
```
cd circuits/node_modules
rm -rf webcola && ln -s ../../webcola/ .
```
Run tests:
```
npm test
```
Serve app:
```
npm start
```
Serve storybook
```
npm run storybook
```
